## üöÄ Getting Started

Follow these instructions to set up the project locally on your machine.

### Prerequisites

* Flutter SDK (Version 3.19.x or higher)
* Dart SDK (Version 3.x.x or higher)
* A code editor (VS Code or Android Studio)

### Installation

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/YourUsername/LedgerPro.git](https://github.com/YourUsername/LedgerPro.git)
    cd LedgerPro
    ```

2.  **Install Dependencies:**
    ```bash
    flutter pub get
    ```

3.  **Run the Application:**
    ```bash
    flutter run
    # To run on Web with HTML Renderer (Recommended for faster loads):
    # flutter run -d chrome --web-renderer html
    ```
